# Tetris

Tetris game build using vanilla javascript, html and css.
Can be used to understand basic Javascript functionality such as:
- innerHTML()
- every()
- some()
- add()
- remove()
- contains()
- addEventListener()
- querySelector()
- querySelectorAll()
- keyCode()
- Math.floor()
- Math.random()
- length
- forEach()
- splice()
- clearInterval()
- setInterval()
